# code-incress
Setup Instructions:

    Install required packages:

    pip install instagrapi

    Update credentials in the script
    Run the script in Kali Linux

Security Notes:

    Use an app-specific password for Instagram
    Be aware that rapid following may trigger account restrictions
    Consider implementing rate limiting to avoid detection
    Monitor Instagram's terms of service compliance

For advanced usage, you could extend this with:

    Proxy rotation to avoid IP bans
    Randomized delays between actions
    Multi-account support
    Error handling for API limits
